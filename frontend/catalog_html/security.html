Робот охранник

Краткое описание

Представляем робот Security, грозный охраник, преданный товарищ и неусыпный вооруженный защитник вашей семьи и дома.

Подробное описание

Представляем робот Security для охраны домов коттеджей и участков.
Вас мучает излишнее любопыство соседей? Постоянная тревога о безопастности сьедает вас?
C роботом SECURITY вы почуствуете надежность и спокойствие, и отстуствие внешнего мира. Требуется оформленная лицензия на ношение оружия
